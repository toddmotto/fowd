Power of jQuery

What jQuery has done for us

What jQuery has done for us

  • Came at the right time
  • Cross-browser consistencies
  • Easy and consistent API
  • Short learning curve
  • Selectors (Sizzle.js)
  • Class manipulation
  • Great event methods
  • Makes difficult stuff a breeze (Ajax/XHR, animation)

JavaScript has listened

What's in store for the future of JavaScript

The time is now

  • Cross-browser consistencies (standards!)
  • Easy and consistent APIs
  • Shorter learning curve
  • Selectors (querySelector!)
  • Class manipulation (classList API)
  • Event standards (addEventListener)
  • Allows JS developers to build micro libs (Ajax/XHR, animation)

Why do I need to learn this stuff?

  • Learn JavaScript, learn library syntax and other libs
  • jQuery cannot provide everything JavaScript has to offer
  • Reliance on jQuery isn't sustainable
  • Learning JavaScript will better you on many levels:
    • Debugging/workarounds
    • Design patterns and structure
    • Career goals/interviews
    • Mastering a language
  • jQuery is often restricting when mixing with JS
  • JavaScript performance far outweighs jQuery's
  • JavaScript is not about the browser anymore!

Class manipulation

jsfiddle.net/toddmotto/Cst3L

// jQuery wraps your code in it's own Object
// which contains the addClass() method
$(elem).addClass('fowd');
      
// JavaScript uses a DOMTokenList to contain
// the class names, returned via the classList Object
elem.classList.add('fowd');

Array looping

jsfiddle.net/toddmotto/2QLwV

var one = [52, 97, 33, 21, 7];
$.each(one, function(index, value) {
    console.log('one:', index, value);
});
      
var two = [52, 97, 33, 21, 7];
two.forEach(function (value, index) {
    console.log('two:', index, value);
});

Object looping

jsfiddle.net/toddmotto/QGyMR

var one = { name: 'Todd', location: 'UK' };
$.each(one, function(index, value) {
    console.log('one:', index, value);
});
      
var two = { name: 'Todd', location: 'UK' };
for (var prop in two) {
    console.log('two:', prop, two[prop]);
}
